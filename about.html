Tecnical Aspects

This site uses Flickr API to search images about an animal species, randomly selected from owl, otter, alpaca, frog and cat and then display a random image selected from the response (you can always go to the original image, otherwhise is a bug, drop in to make an issue).

jQuery is used for manipulation and animations of the page. Crossroad.js + Hasher is used for the fancy URLs action.

You can find the icons at Game-Icons, and the spinning infinite icon is from Preloaders

If you are curious you can find the source code on github
